Introduction

Salutation à vous, programmeurs en devenir !
Tu débutes la programmation et tu es intéressé par le python ? N’aie crainte ! Grâce à ces cours tu vas pouvoir apprendre tout ce dont tu a besoin pour pouvoir commencer à programmer en python.

Pourquoi apprendre le Python ?

Le python est un langage de programmation facile à apprendre notamment en le pratiquant.
Il bénéficie également d’une lisibilité bien plus importante que la plupart des autres langages de programmation. En effet quand l’on observe un code en python on peut voir que pour faire un programme il faut beaucoup moins de ligne de code que pour un autre langage.

Par exemple le programme basique : « Hello World »

Pour faire un programme qui fait exactement la même chose on a du écrire beaucoup plus de ligne de codes.

Python c’est aussi un langage polyvalent et open source en plus de pouvoirs fonctionner sur plusieurs plateforme, que ce soit Linux , Windows ou Mac OS . Cela permet aux initiés de trouver de l’emploi plus facilement car c’est une compétence très recherchée par les employeurs.
Il est donc temps pour vous d’apprendre ce langage merveilleux !

À vos claviers !
