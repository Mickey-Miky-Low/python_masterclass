Les modules extérieur

Ils existe plein de modules comprenant des fonctions autour des mathémaiques (math), itinérable (itertools), tableau à n dimantion (numpy) ou bien permettant l'interaction avec l'os (os) ou le systeme (sys).

import itertools import numpy as np from math import cos from math import *

Tout d'abord il existe plusieurs façons d'importer des modules. A la première ligne nous importons tout le module itertools, à la deuxièeme ligne on import le module numpy en le renommant via le mot clé as permettant de racoursir l'ecriture. A la troisème ligne nous importons que la fonction cos et nous n'auront donc plus a préciser le module lors de l'appel. Pour la dernière ligne elle permet d'importer toute les fonctions d'un modules sans avoir repriser le module.

# parcours tout les tulpes possible composé de 0123456789abcdef d'une longeur de 2 for chars in itertools.product("0123456789abcdef", repeat=2): print("".join(chars)) # crée une matrice de 5x9 array = np.zeros((5, 9)) print(array) # cosinus de 10 print(cos(10))

La puissance des modules est que vous n'avez pas refaire ce qui a déjà été partager par d'autre. Cela permet de faire des projets rapidement et efficasement et fait une des force majeurs de python.

Vos propres modules

Vos code commancent à être long il serais donc mieux de pouvoir les mettres dans plusieurs fichier et bien vous le pouvez et ceux d'un façons très simple .

def bonjour(nom): print(f"Bonjour {nom}")

Voulais un code simple que vous aimeriez utilisé dans votre programme principale, vous avez juste à faire comme suit.

from salutation import bonjour bonjour("Alice") # Bonjour Alice

Pour que tout ce passe comme prévu il faut bien que les deux fichier soit au endroit

Dans le cas ou il salutation serais dans un dossier nommé mesfonction il faudrait non plus import salutation mais mesfonction.salutation
